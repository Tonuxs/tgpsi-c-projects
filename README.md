# TGPSI C Projects

A collection of C programming exercises and projects developed during the TGPSI (Technician in Management and Programming of Information Systems) course at Agrupamento de Escolas da Batalha.

## 📖 About

This repository documents my journey learning the C programming language through practical exercises and academic projects. The goal is to strengthen fundamental programming skills, problem-solving abilities, and logical thinking.

## 🎯 Objectives

* Practice programming fundamentals
* Develop problem-solving skills
* Learn algorithm design and implementation
* Improve code organization and debugging techniques
* Build a solid foundation for future software development

## 🛠️ Topics Covered

* Variables and data types
* Conditional statements
* Loops and iteration
* Functions
* Arrays and strings
* Pointers
* Structures
* File handling
* Basic algorithms

## 📂 Repository Structure

Each folder contains exercises or projects related to a specific topic studied during the course.

```text
Topic/
├── exercise1.c
├── exercise2.c
└── project.c
```

## 🚀 How to Run

Compile a program using GCC:

```bash
gcc filename.c -o program
./program
```

Example:

```bash
gcc calculator.c -o calculator
./calculator
```
